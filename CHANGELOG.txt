SCHIMBARI
Versiune curenta: 1.3.3

Versiunea 1.3.3
- Corectie build: scriptul foloseste prioritar `.venv\Scripts\python.exe`, cu fallback la `venv\Scripts\python.exe`.
- Corectie installer: instalarea este fortata pe arhitectura 64-bit (`ArchitecturesAllowed` si `ArchitecturesInstallIn64BitMode` setate la `x64compatible`).
- Corectie update installer: nu se mai mosteneste automat folderul vechi de instalare (`UsePreviousAppDir=no`), evitand reinstalari in `Program Files (x86)`.

Versiunea 1.3.2
- Update silent: aplicatia este relansata automat dupa finalizarea instalarii.
- Corectie startup: focusul initial este setat pe campul "Furnizor", nu pe "Minim".
- Corectie build: iconitele SVG pentru butoanele controlului "Comanda minima" sunt incluse corect in pachet.

Versiunea 1.3.1
- Ajustari pentru controlul "Comanda minima": stilizare pentru spin buttons si intervale de valori actualizate.
- Iconite SVG dedicate pentru butoanele de incrementare/decrementare.

Versiunea 1.3.0
- Filtru nou "Comanda minima" in UI, cu functionalitate de filtrare asociata.
- Meniu contextual in tabele pentru copiere rapida a continutului celulelor.
- Refactor pentru verificarea/update-ul SHA-256 din release-urile GitHub.
- Scriptul de build nu mai genereaza checksum local si nu mai sterge .gitignore la cleanup.
- Teste extinse pentru fluxuri si helper-e.

Versiunea 1.2.0
- Cautare avansata: suport expresii multi-termen cu AND/OR/NOT.
- Interfata de cautare extinsa cu campuri suplimentare, reset imbunatatit si calcul mai bun al inaltimii randurilor.
- Dialoguri AutoCmd si AutoLivrare refactorizate (baza comuna ThresholdParamsDialog) si flux de export consolidat.
- Update instalator securizat: generare si verificare checksum SHA-256 la descarcare.
- Stabilitate la inchidere: oprire corecta pentru thread-urile de update si download.
- Gestionare stare de incarcare imbunatatita si fallback mai robust pentru import ctypes.
- Teste unitare noi pentru helper-e, compararea versiunilor si extragerea SHA-256.
- Dependinte actualizate in requirements.txt.

Comparatie cu ultimul release (1.0.1)
- Functionalitati noi majore: AutoCmd/AutoLivrare + cautare avansata.
- Securitate si fiabilitate crescute pentru update-ul aplicatiei.
- Acoperire de testare extinsa semnificativ.

Versiunea 1.0.1
- Rezolvat: meniul "Locatie baza" nu mai ramane blocat/greyed-out.

Versiunea 1.0.0
- Prima versiune publica.
