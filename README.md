

# RstocMag — Descarcari

RstocMag este o aplicatie desktop, pentru analiza stocurilor si generarea comenzilor intre locatii, pe baza unui raport de gestiune, generat din aplicatii terte.

Acest repository este utilizat exclusiv pentru distributia aplicatiei (binar Windows). Codul sursa este privat.

---

## Descarcare

Accesati sectiunea **Releases** si descarcati ultima versiune disponibila (fisier .zip).

---

## Cerinte sistem

- Windows 10 / 11 (64-bit)

---

## Utilizare

1. Generati din programul de gestiune raportul:
   „Export excel stoc pe lant la data”.
2. Deschideti RstocMag.
3. Apasati „Reincarca”.
4. Introduceti comenzile dorite.
5. Exportati comenzile.
